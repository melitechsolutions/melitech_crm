QUICK START
================================================================================

Windows:
  1. Extract melitech-crm-windows.zip
  2. Double-click SETUP-WINDOWS.bat
  3. Double-click START-DEV-WINDOWS.bat
  4. Open http://localhost:5173

Linux:
  1. tar -xzf melitech-crm-linux.tar.gz
  2. cd melitech-crm
  3. ./setup-linux.sh
  4. ./start-dev-linux.sh
  5. Open http://localhost:5173

Docker (Windows):
  1. Extract melitech-crm-windows.zip
  2. Ensure Docker Desktop is running
  3. Double-click SETUP-DOCKER-WINDOWS.bat
  4. Open http://localhost:5173

Docker (Linux):
  1. tar -xzf melitech-crm-linux.tar.gz
  2. cd melitech-crm
  3. ./setup-docker-linux.sh
  4. Open http://localhost:5173

PRODUCTION DEPLOYMENT
================================================================================

Before deploying:
  1. Update .env with production values
  2. Change JWT_SECRET to strong random value
  3. Enable HTTPS
  4. Update CORS settings
  5. Set NODE_ENV=production
  6. Run npm run build

For Docker:
  docker-compose -f docker-compose.prod.yml build
  docker-compose -f docker-compose.prod.yml up -d
