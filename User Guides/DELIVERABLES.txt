QUICK START COMMANDS (WINDOWS)
================================================================================

1. Extract the archive:
   Expand-Archive -Path melitechcrm-updated.zip -DestinationPath .

2. Navigate to project:
   cd melitechcrm

3. Install dependencies:
   pnpm install

4. Build the application:
   pnpm build

5. Start the server:
   pnpm start

6. Open browser:
   http://localhost:3000
